A memory chip in Voyager 1’s flight data subsystem (FDS), which encodes engineering data, failed, stopping data transmission. NASA engineers at the Jet Propulsion Laboratory (JPL) fixed the issue by changing FDS memory codes, allowing Voyager 1’s health and status data to reach Earth again.

On March 15, JPL engineers carried out a comprehensive FDS memory readout to find the problem. Given the spacecraft’s distance—15 billion miles (24 billion kilometers)—replacing the chip was impossible.
